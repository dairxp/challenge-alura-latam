# Challenge Data Science

## Analisis de Ventas de Tiendas

### Requisitos

- Python
- Pandas
- Matplotlib

### Uso

1. Clona el repositorio principal:
   ```sh
   git clone https://github.com/dairxp/challenge-alura-latam
   ```
2. Entra a la carpeta del proyecto:
	```sh
	cd challenge-alura-latam/challenge-data-science-latam
	```
3. Abre el notebook [AluraStoreLatam.ipynb](AluraStoreLatam.ipynb).
