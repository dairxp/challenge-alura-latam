# Amigo Secreto

## Descripcion

Aplicacion web del juego "Amigo Secreto". Permite agregar nombres a una lista y sortear un amigo secreto de forma aleatoria.

## Caracteristicas

- Agregar nombres de amigos a una lista.
- Validar que los nombres no esten vacios.
- Mostrar la lista de amigos agregados.
- Seleccionar aleatoriamente un amigo secreto.
- Mostrar el resultado del sorteo.

## Instalacion

1. Clona el repositorio principal:
   ```sh
   git clone https://github.com/dairxp/challenge-alura-latam
   ```
2. Entra a la carpeta del proyecto:
   ```sh
   cd challenge-alura-latam/challenge-amigo-secreto
   ```
3. Abre el archivo [index.html](index.html) en tu navegador.

## Estructura

- [index.html](index.html): interfaz principal.
- [style.css](style.css): estilos.
- [app.js](app.js): logica del sorteo.
- [assets/](assets/): recursos del proyecto.
